# DSA Essentials Landing Page
A modern and responsive landing page built using **HTML and CSS**, inspired by real-world ed-tech platforms. This project focuses on clean UI design, proper layout structuring, and precise spacing using Flexbox.

##  Features
- Modern landing page layout
- Responsive navigation bar and footer
- Structured sections for content clarity
- Clean alignment using Flexbox
- Well-organized and readable CSS

##  Tech Stack
- HTML5
- CSS3

This project was created as a **frontend practice exercise** to improve layout design skills, spacing control, and real-world website structuring using pure HTML and CSS.
